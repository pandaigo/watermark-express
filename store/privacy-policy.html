Privacy Policy

Last updated: 2026-05-03

Watermark Express ("the Extension") is a Chrome browser extension that adds text or logo watermarks to images locally in the browser.

Data Collection

The Extension does not collect, transmit, or share any personal data. Specifically:

  • No data is sent to any external server.
  • No analytics, tracking, or telemetry is included.
  • No user accounts are required.
  • No cookies are set by the Extension.

Image Processing

All image processing (watermark application) is performed entirely within the browser using the HTML5 Canvas API. Your images are never uploaded to any server. They remain on your device at all times.

Data Storage

All user data and settings (watermark preferences, templates, logo images) are stored locally on the user's device using the Chrome Storage API (chrome.storage.local). This data never leaves the device.

Permissions

The Extension requests the following permissions:

  • storage — To save user settings, watermark templates, and payment status locally on the device.

Third-Party Services

The Extension uses ExtensionPay (extensionpay.com) to process one-time payments and verify purchase status. When a user upgrades to the Pro version, payment is handled by ExtensionPay via Stripe. No user data or personal information is shared with these services beyond what is necessary for payment processing.

Changes to This Policy

If this policy is updated, the changes will be posted on this page with an updated date.
